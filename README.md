# Flask-Car
> Put the HTML file in a folder named templates
RC car controlled by a flask webpage, using raspberry pi 4 GPIO pins

I used:
1. L298n
2. 4 DC motors
3. Jumper wires
4. Raspberry pi 4
5. Battery pack, 6 AA batteries
